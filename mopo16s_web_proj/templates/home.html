    One of the key tools for studying microbial diversity of a site is the targeted amplicon sequencing of an informative marker, such as the ribosomal small subunit (16S ribosomal DNA). The accuracy of targeted amplicon sequencing strongly depends on the choice of primer pairs.
    mopo16S offers an automated method for optimal multi-objective bacterial 16S primers design.


    What can this app do?
    This web application provides access to mopo16S functions through an easy-to-use graphic interface. The purpose is to allow the execution of the calculations in a cloud environment and to provide data analytics tools.


    1) Upload your sequence set or check if existing databases are suitable for you.
    View existing sequences Create new sequence set

    2) Upload your primer set.
    View existing primers Create new primer set

    3) Submit your job defining mopo16S parameters.
    View submitted jobs Create new job

    4) View job results with plots and tables.
    View results
